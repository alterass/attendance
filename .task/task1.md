# 实现一个考勤记录器
该应用目的是为了记录和统计加班、调休、请假，方便迁入到第三方系统，数据只存在本地indexeddb里

## 页面

### 注册
注册用户页面，字段包含姓名、部门、初始可调休时长；除了注册页面外，还包含一个导入功能，导入自动注册
数据结构如下：
```ts
interface User {
  /** 姓名 */
  name: string
  /** 部门 */
  department: string
  /** 可调休时长 */
  offHour: number
}
```

### 首页
- 个人信息，如姓名、部门
- 导出按钮，可导出自己的所有的数据
- 当月加班、调休、请假统计
- 新增考勤记录按钮

### 考勤记录表单视图
考勤记录详情、新增、编辑共用页面，分新增、只读、编辑状态；当新增成功后，跳到成功界面，成功界面包含当次新增的信息、返回按钮、继续新增按钮；
数据结构如下：
```ts
interface AttenddanceRecord {
  /** primary id */
  id: string
  /** 考勤类型 */
  type: "加班" | "调休" | "请假"
  /** 具体事由 */
  desc: string
  /** 起始时间 */
  beginTime: number
  /** 结束时间 */
  overTime: number
  /** 创建时间 */
  createAt: number
  /** 更新时间 */
  updateAt: number
}
```

### 考勤记录列表视图
- 筛选：日期范围、类型
- 列表
- 新增按钮
- 勾选功能
- 生成考勤申请按钮，使用Dropdown实现，数据源可以是勾选，本月、本周、上周
